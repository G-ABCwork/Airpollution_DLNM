# 대기오염원 건강영향평가

- 질병관리청 발주
- [학술연구] 빅데이터를 이용한 미세먼지 건강영향평가
- 2021.  03 -

## 평가 질환

-   신경과 질환
-   정신과 질환
-   안과 질환
-   호흡기 질환

-   Distributed lag non linear model

-   R
